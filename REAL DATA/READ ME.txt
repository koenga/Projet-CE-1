There are four main files: 

- Main_Gather_Pretrain_Data_GRU.ipynb
- Main_pretrain_GRU.ipynb 
- Main.ipynb 
- Main_plotting.ipynb

The ones you are interested in are the two last.

To make the Main.ipynb work you need to: 
- change the paths to load the data: namely variables 'link' and 'df_link'
- the path for 'link' should lead to all the pickle files
- the path for 'df_link' is used to open the file 'link_bboxes_clustered.csv'

After you changed that everything should be running.

You change choose the mode, with the variable called mode in the run function.
You have the choice between: 'random', 'gru', and 'greedy'.

To plot the results, you can use Main_plotting.ipynb. Be sure to change the path to plot 
the results you are interested in.

The current results are the last one we sent you, with recover_param = 0 for all three modes.
